Documentation UI Library EQUITECH

Démarrage

Bienvenue dans la documentation de la UI Library EQUITECH

La UI Library EQUITECH est une bibliothèque de composants React moderne et accessible, conçue pour créer des interfaces utilisateur cohérentes et professionnelles.

Caractéristiques principales

  • Design System cohérent : Palette de couleurs et typographie unifiées
  • Composants accessibles : Respect des standards WCAG et ARIA
  • Responsive Design : Adaptation automatique à tous les écrans
  • TypeScript Ready : Support complet des types TypeScript
  • Performance optimisée : Composants légers et rapides
// Installation
npm install @equitech/ui-library

// Import d'un composant
import { Button } from '@equitech/ui-library';

// Utilisation
<Button variant="primary">Cliquez-moi</Button>

Installation

Guide d'installation et configuration

Installation via npm

npm install @equitech/ui-library

Import des styles

// Dans votre fichier principal
import '@equitech/ui-library/dist/index.css';

Configuration TypeScript

// tsconfig.json
{
  "compilerOptions": {
    "moduleResolution": "node",
    "esModuleInterop": true,
    "allowSyntheticDefaultImports": true
  }
}
