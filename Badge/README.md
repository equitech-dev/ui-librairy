# Badge

Badge/statut universel, personnalisable.

## Props
- `label` : texte affiché
- `type` : default | success | warning | error | info
- `className`, `style` : personnalisation
- `aria-label` : accessibilité

## Exemple
```jsx
<Badge label="Nouveau" type="info" />
```

## Accessibilité
- Peut être lu par les lecteurs d'écran
